Copia esta estructura como punto de partida:

{
  "title": "Nombre de tu trivia",
  "time_per_question": 20,
  "questions": [
    {
      "question": "¿Cuál es la capital de Perú?",
      "options": ["Lima", "Cusco", "Arequipa", "Trujillo"],
      "correct": 0
    }
  ]
}

correct es el número de posición (empezando en 0) de la opción correcta dentro de options. Puedes incluir tantas preguntas como quieras dentro de la lista.
